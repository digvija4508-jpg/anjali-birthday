# Site Vision: Happy Birthday Didi 🎂

## 1. Purpose
A single-page birthday celebration website for the user's older sister ("Didi"). Themed around Lilo & Stitch with a hand-drawn crayon aesthetic. Features interactive elements so visitors can leave birthday wishes.

## 2. Target
- **Birthday Date:** April 24, 2026 at 12:00 PM IST
- **Audience:** Family and friends visiting to celebrate
- **Tone:** Warm, funny, loving, playful

## 3. Key Features
- [x] Hero section with CSS Stitch character and birthday greeting
- [x] Countdown timer to the birthday moment
- [x] Heartfelt birthday letter
- [x] "Why You're Amazing" fun cards
- [x] Interactive sticky notes wall (localStorage)
- [x] Birthday wish generator
- [x] Balloon pop mini-game
- [x] Stitch-themed footer

## 4. Sitemap
- [x] `index.html` — Single-page site with all sections

## 5. Technology
- Pure HTML + CSS + JavaScript
- No frameworks, no build tools
- Google Fonts (Fredoka, Caveat, Patrick Hand)
- CSS art for Stitch character (no images)
- localStorage for sticky notes persistence
